ğŸ“‹ STEP-BY-STEP SETUP

  1. Create Conda environment (one time)
     conda env create -f environment.yml

  2. Activate
     conda activate vrswap

  3. Download model
     inswapper_128.onnx (from HuggingFace)
     Place in project root folder

  4. Create faces folder
     mkdir faces
     Put your JPG/PNG files there

  5. Run!
     python process_video.py --video myvideo.mp4 --faces ./faces --output result.mp4
